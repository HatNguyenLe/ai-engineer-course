01
Nền tảng & Toán học
Python · Toán học · Dữ liệu cơ bản
4–6 tuần ▶
Đạt được: Tự tin viết code Python, xử lý dữ liệu cơ bản và hiểu các khái niệm toán học cốt lõi đứng sau AI — không còn bị "mù" trước thuật ngữ.

Nội dung trọng tâm

  • Python nâng cao: NumPy, Pandas, Matplotlib — xử lý mảng, DataFrame, vẽ biểu đồ
  • Đại số tuyến tính: Vector, ma trận, phép nhân ma trận, eigenvalue — trực quan hóa bằng code
  • Xác suất & thống kê: Phân phối, Bayes, trung bình / phương sai, độ lệch chuẩn
  • Giải tích cơ bản: Đạo hàm, chain rule — nền tảng cho backpropagation
  • Git & môi trường làm việc: Conda / venv, Jupyter Notebook, GitHub workflow

Dự án thực hành

Bài tập
Phân tích bộ dữ liệu Titanic — làm sạch dữ liệu, vẽ 5 biểu đồ thống kê có ý nghĩa, viết báo cáo ngắn bằng Jupyter Notebook. Xem hướng dẫn chi tiết bên dưới.
02
Machine Learning cổ điển
Scikit-learn · Thuật toán · Đánh giá mô hình
5–7 tuần ▶
Đạt được: Xây dựng và đánh giá các mô hình ML cổ điển, hiểu khi nào dùng thuật toán nào, biết phát hiện overfitting và cách xử lý.

Nội dung trọng tâm

  • Supervised learning: Linear/Logistic Regression, Decision Tree, Random Forest, SVM, KNN
  • Unsupervised learning: K-Means clustering, PCA giảm chiều dữ liệu
  • Feature engineering: Xử lý missing values, encoding, scaling, tạo feature mới
  • Đánh giá & tuning: Cross-validation, confusion matrix, ROC-AUC, GridSearchCV
  • MLOps cơ bản: Lưu/load model với pickle, pipelines với scikit-learn

Dự án thực hành

Bài tập
Dự đoán giá nhà (Boston / Ames Housing) — so sánh ít nhất 3 thuật toán, vẽ learning curve, tối ưu hyperparameter, báo cáo model tốt nhất kèm giải thích.
03
Deep Learning & Neural Networks
PyTorch · CNN · RNN · Training loop
7–10 tuần ▶
Đạt được: Xây dựng, train và debug neural network từ đầu với PyTorch. Hiểu bản chất của backprop, gradient descent và các kiến trúc phổ biến CNN / RNN.

Nội dung trọng tâm

  • PyTorch fundamentals: Tensor, autograd, custom Dataset, DataLoader, training loop đầy đủ
  • CNN: Convolution, pooling, kiến trúc ResNet — áp dụng cho Computer Vision
  • RNN / LSTM / GRU: Sequence modeling, time-series, xử lý văn bản đơn giản
  • Kỹ thuật training: Batch norm, dropout, learning rate scheduler, early stopping, weight init
  • Thực hành GPU: Huấn luyện trên Google Colab / Kaggle GPU, đọc TensorBoard logs

Dự án thực hành

Bài tập
Phân loại ảnh CIFAR-10 với CNN tự thiết kế — đạt ≥ 85% accuracy, implement data augmentation, vẽ training/validation curve, so sánh với baseline MLP.
04
LLM & Generative AI
Transformer · Fine-tuning · RAG · Prompt Engineering
8–12 tuần ▶
Đạt được: Làm chủ Transformer architecture, fine-tune LLM cho bài toán cụ thể, xây dựng hệ thống RAG và triển khai AI chatbot hoàn chỉnh.

Nội dung trọng tâm

  • Transformer & Attention: Self-attention, multi-head attention, positional encoding, kiến trúc BERT / GPT
  • Hugging Face ecosystem: Transformers, Datasets, PEFT (LoRA, QLoRA) để fine-tune hiệu quả
  • RAG: Vector database (Chroma, Pinecone), embedding models, LangChain / LlamaIndex
  • Prompt Engineering: Few-shot, chain-of-thought, structured output, function calling với API
  • Evaluation LLM: BLEU, ROUGE, LLM-as-judge, human evaluation workflow

Dự án thực hành

Bài tập
Xây dựng chatbot hỏi đáp tài liệu nội bộ — upload PDF, chunk + embed vào vector DB, truy vấn bằng LLM, đánh giá độ chính xác trên 20 câu hỏi test.
05
Production & AI Systems
MLOps · Deployment · Monitoring · System Design
8–10 tuần ▶
Đạt được: Triển khai AI system lên production, xây dựng CI/CD pipeline cho ML, monitor model drift, thiết kế hệ thống AI scalable — sẵn sàng đảm nhận vai trò AI Engineer thực chiến.

Nội dung trọng tâm

  • Model serving: FastAPI, BentoML hoặc Triton Inference Server — REST / gRPC endpoint, batching
  • MLOps pipeline: MLflow experiment tracking, DVC data versioning, GitHub Actions CI/CD tự động
  • Containerization & Cloud: Docker, Kubernetes cơ bản, deploy lên AWS / GCP / Azure, auto-scaling
  • Monitoring & observability: Detect data/model drift, Prometheus + Grafana, A/B testing strategy
  • AI System Design: Latency vs accuracy trade-off, caching, async inference, cost optimization

Dự án capstone

Capstone
Deploy end-to-end ML system — từ training (MLflow tracking) → Docker container → API endpoint → Kubernetes cluster → dashboard monitor model drift theo thời gian thực.

Load & khám phá dữ liệu
Đọc file CSV, xem cấu trúc, thống kê tổng quan

import pandas as pd import numpy as np import matplotlib.pyplot as plt import seaborn as sns # Load từ seaborn (không cần download) df = sns.load_dataset('titanic') # Xem 5 hàng đầu df.head()

🧹
Làm sạch dữ liệu
Xử lý missing values, sửa kiểu dữ liệu, loại duplicates
Titanic dataset có ~20% giá trị null ở cột age và gần 80% null ở deck. Đây là cơ hội học cách xử lý missing data thực tế.

Kiểm tra missing values

# Đếm null theo từng cột missing = df.isnull().sum() missing_pct = (missing / len(df) * 100).round(1) missing_df = pd.DataFrame({ 'missing_count': missing, 'missing_pct': missing_pct }).sort_values('missing_pct', ascending=False) print(missing_df[missing_df.missing_count > 0])

Xử lý missing values

# Fill tuổi bằng median theo từng hạng vé (pclass) df['age'] = df.groupby('pclass')['age'].\ transform(lambda x: x.fillna(x.median())) # Fill embarked bằng mode df['embarked'].fillna(df['embarked'].mode()[0], inplace=True) # Drop cột quá nhiều null (>70%) df.drop(columns=['deck'], inplace=True) print(f"Missing sau xử lý: {df.isnull().sum().sum()}")
✓ Luôn giải thích tại sao chọn cách fill đó (median vs mean vs mode) — đây là điều người review sẽ hỏi đầu tiên.

Phân tích thống kê
Tỉ lệ sống sót, so sánh nhóm, tương quan
Mục tiêu: tìm ra yếu tố nào ảnh hưởng đến khả năng sống sót. Đặt câu hỏi trước khi code, đừng code mò.

Phân tích tỉ lệ sống sót

# Tổng quan tỉ lệ sống sót survival_rate = df['survived'].mean() * 100 print(f"Tỉ lệ sống sót tổng: {survival_rate:.1f}%") # Theo giới tính print("\nTheo giới tính:") print(df.groupby('sex')['survived'].mean().round(3) * 100) # Theo hạng vé print("\nTheo hạng vé (pclass):") print(df.groupby('pclass')['survived'].mean().round(3) * 100)

Ma trận tương quan

# Tương quan giữa các biến số numeric_cols = ['survived', 'pclass', 'age', 'sibsp', 'parch', 'fare'] corr_matrix = df[numeric_cols].corr().round(2) print(corr_matrix)

5 câu hỏi cần trả lời trong báo cáo

  • Phụ nữ có tỉ lệ sống sót cao hơn đàn ông không? Chênh lệch bao nhiêu %?
  • Hành khách hạng 1 có lợi thế gì so với hạng 3?
  • Độ tuổi trung bình của người sống sót vs không sống sót là bao nhiêu?
  • Giá vé (fare) có tương quan với tỉ lệ sống sót không?
  • Cảng khởi hành (Cherbourg/Southampton/Queenstown) có ảnh hưởng không?
